# FizzBuzz

.NET implementation of the FizzBuzz text transformation task.

## Build

```bash
dotnet build
```

## Test

```bash
dotnet test
```

## Rules

- Every 3rd word -> `Fizz`
- Every 5th word -> `Buzz`
- Every 15th word -> `FizzBuzz`
- Only alphanumeric words are counted
- Punctuation and whitespace are preserved
